# uoftverse

Take your first step into UofTverse; a webbase platform designed for UofT students and members of faculty alike. In this centralized hub, members of the UofT community can share and find opportunities around campus including research, lab positions, and design teams. 

Employing Python and the use of AWS services such as Lambda, DynamoDB, and AI tools similar to Bedrock, UofTVerse builds a user database and matches users with opportunities around campus by analyzing user queries for keywords, profiles, departements, and specific types. 

The frontend is built using JavaScript, React, Node.js, Leaflet, HTML, and CSS, and features profile displays and our key feature Into the UofTVerse: a map visualization tool that geographically represents opportunities around campus, guiding the users to the buildings relating the departments they're interested in. 

Future development plans include using AWS Amplify to deploy the website, AWS Location Services to support the map tool, and Amazon Cognito for user verification to ensure our platform remains safe and reliable for members of the UofT community.
